# Deskflow UI 功能需求文档

## 文档概述

本文档详细描述 Deskflow 应用的 UI 功能需求，包括快速笔记页、桌面美化组件、脚本和笔记管理页等核心功能模块。

---

## 1. 快速笔记页

### 1.1 功能概述

快速笔记页是用户最常用的功能，提供轻量级、快速访问的笔记编辑体验，支持 Markdown 语法和实时预览。

### 1.2 核心功能

#### 1.2.1 笔记编辑器

```mermaid
graph TB
    A["笔记编辑器"] --> B["基础编辑"]
    A --> C["Markdown 支持"]
    A --> D["自动保存"]
    A --> E["快捷操作"]

    B --> B1["文本输入"]
    B --> B2["撤销/重做"]
    B --> B3["复制/粘贴"]
    B --> B4["全选"]

    C --> C1["标题"]
    C --> C2["列表"]
    C --> C3["代码块"]
    C --> C4["链接"]
    C --> C5["图片"]
    C --> C6["表格"]
    C --> C7["引用"]

    D --> D1["防抖保存 500ms"]
    D --> D2["自动创建标题"]
    D --> D3["时间戳更新"]

    E --> E1["快捷键支持"]
    E --> E2["工具栏"]
    E --> E3["右键菜单"]
```

**功能需求**：

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 文本编辑 | 支持多行文本输入，自动换行 | P0 |
| Markdown 语法 | 支持常用 Markdown 语法 | P0 |
| 自动保存 | 防抖 500ms 后自动保存 | P0 |
| 标题提取 | 从第一行自动提取标题 | P0 |
| 撤销/重做 | 支持 Ctrl+Z/Ctrl+Y | P1 |
| 快捷键 | 支持 Ctrl+B 加粗、Ctrl+I 斜体等 | P1 |
| 代码高亮 | 代码块语法高亮 | P2 |
| 实时预览 | Markdown 实时预览 | P2 |

#### 1.2.2 标签页管理

```mermaid
graph LR
    A[标签页管理] --> B[新建标签]
    A --> C[切换标签]
    A --> D[关闭标签]

    B --> B1[创建新笔记]
    B --> B2[打开现有笔记]

    C --> C2[快捷键切换]

    D --> D2[关闭其他标签]
    D --> D3[关闭所有标签]
    
    E --> E1[未保存标记 *]
    E --> E2[标题显示]
    E --> E3[图标显示]
```

**功能需求**：

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 多标签页 | 支持同时打开多个笔记 | P0 |
| 新建标签 | 点击 + 按钮创建新笔记 | P0 |
| 切换标签 | 点击标签切换笔记 | P0 |
| 关闭标签 | 点击 × 关闭标签 | P0 |
| 未保存标记 | 未保存的标签显示 * 标记 | P1 |
| 标签拖拽 | 支持拖拽排序标签 | P2 |
| 标签右键菜单 | 右键显示操作菜单 | P2 |
